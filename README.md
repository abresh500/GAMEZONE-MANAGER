GAMEZONE MANAGER

well its  simple PlayStation gaming center management system built with Java for a school project.
You can start and end sessions for customers, track how many games they played, calculate the total price, and generate a daily report. Everything gets saved to a local database.
Regular customers pay 25 birr per game and discount customers pay 20 birr. The total is calculated at the end of the session when you enter how many games were played.
To run it, open the project in IntelliJ IDEA, make sure the SQLite JDBC driver is added to the libraries, and run Main.java.
